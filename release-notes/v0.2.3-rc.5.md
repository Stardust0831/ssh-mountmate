# SSH MountMate v0.2.3-rc.5

This prerelease focuses on UI regression fixes after the previous readability changes.

## Changes

- Reworked the Settings window so its content scrolls while the save button stays visible.
- Fixed Add config scroll handling so scrolling the main page no longer scrolls an open dialog.
- Reduced excessive label/input spacing in Add config.
- Restored more natural Tk font sizing instead of forcing global DPI/font sizes.
- Changed batch SSH config import actions to checkboxes: new configs are checked for import, duplicates are skipped unless overwrite is checked.
- Added "Import all new" and "Overwrite all duplicates" controls for batch SSH config import.
- Simplified batch import rows so Import and Overwrite stay aligned without the extra table-like header.
- Kept required-field markers in a fixed slot so Add config input fields line up across source modes.
- Added question-mark help icons in Add config instead of binding long tips directly to controls.
- Cleaned up SSH config help text and the comments written to app-managed SSH config files.
- Improved tooltip positioning and width so the cursor is less likely to cover help text.
