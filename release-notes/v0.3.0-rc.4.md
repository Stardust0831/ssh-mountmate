# SSH MountMate v0.3.0-rc.4

Pre-release fixing Explorer refresh at Windows drive roots.

## Changes

- Prevent a drive-root path such as `Y:\` from leaving a literal quote in the `--refresh-path` argument.
- Register Explorer refresh commands with a trailing `\.` path component so Windows command-line parsing remains correct for both drive roots and regular directories.
- Repair malformed trailing-quote paths produced by context menus registered by earlier release candidates.
- Normalize root-directory refreshes to the empty rclone remote path before calling `operations/list`.

## Verification

- 71 unit tests pass, including drive-root quote repair and root remote-path normalization.
- Windows `CommandLineToArgvW` verification reproduces the old `Y:"` result and confirms the new argument normalizes to `Y:\`.
- Source compilation and whitespace checks pass.

# SSH MountMate v0.3.0-rc.4 中文说明

这个预发布版修复 Windows 盘符根目录的资源管理器右键刷新。

## 变化

- 避免 `Y:\` 这类盘符根路径在 `--refresh-path` 参数中遗留字面量引号。
- 注册资源管理器刷新命令时追加 `\.` 路径组件，使 Windows 命令行在盘符根目录和普通文件夹中都能正确解析。
- 兼容并修复旧候选版本右键菜单可能产生的末尾引号路径。
- 调用 `operations/list` 前，将挂载根目录刷新标准化为空的 rclone 远端路径。

## 验证

- 71 个单元测试通过，包括盘符根目录引号修复和根远端路径标准化测试。
- Windows `CommandLineToArgvW` 验证复现了旧参数的 `Y:"` 结果，并确认新参数最终标准化为 `Y:\`。
- 源码编译和空白检查通过。
