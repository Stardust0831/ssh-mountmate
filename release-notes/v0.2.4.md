# SSH MountMate v0.2.4

This release focuses on high-DPI UI clarity, mount-card readability, and safer SSH config batch import behavior.

## Changes

- Improve Windows high-DPI rendering and default window sizing.
- Increase readability of toolbar buttons, checkboxes, help icons, dialogs, and mount-card text.
- Refine mount-card layout: local mount path, capacity usage, remote path, and user host now occupy clearer positions.
- Improve the capacity bar so remote paths fit better and descenders remain visible.
- Add status badges to SSH config batch import rows for `NEW`, `SAME`, `SAME HOST`, `SAME TARGET`, and `INVALID`.
- Highlight changed fields in batch import details when comparing imported and saved configs.
- Disable batch overwrite for saved configs that are currently mounted or busy; unmount them from the main window before overwriting.
