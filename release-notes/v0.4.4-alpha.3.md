# SSH MountMate v0.4.4-alpha.3

This prerelease streamlines saved-order and login-startup controls.

- **Adjust saved order** is now an action inside the sort menu instead of a separate prominent
  button. The redundant fixed-width wrapper was removed to eliminate the unexplained empty space.
- Batch mode now shows a separate **At login** checkbox on every eligible connection card instead
  of one aggregate checkbox for all selected connections.
- Interactive SSH connections are identified as requiring interactive login and do not expose an
  auto-mount checkbox that cannot work without a live session.

---

本预发布版简化了保存顺序与登录自启的操作方式。

- “调整保存顺序”现在作为排序下拉菜单中的一个操作项，不再使用单独醒目的按钮；同时移除了多余的
  固定宽度包装，消除排序控件旁意义不明的空白。
- 批量模式现在在每个支持自启的连接卡片中显示独立的“登录自启”复选框，不再用一个总复选框控制
  所有选中连接。
- 交互式 SSH 连接会标明需要交互式登录，不再展示必须依赖实时会话、实际无法生效的自动挂载选项。
