# SSH MountMate v0.2.1-rc.7

Pre-release build for Windows stale mount status detection.

## Fixes

- Allows mount status to fall back to `PID exists + mountpoint is ready` when Windows cannot return the rclone command line.
- Keeps command-line matching strict when the command line is available, so unrelated processes are still rejected.

## Verification

- `python3 -m py_compile $(find src build tests -name '*.py' -print) launcher.py`
- `PYTHONPATH=src python3 -m unittest discover -s tests -v`
