# SSH MountMate v0.2.1-rc.8

Pre-release build for GUI refresh and capacity query performance.

## Fixes

- Adds a short TTL cache for capacity checks so mounted configs do not run `rclone about` on every refresh.
- Prevents overlapping capacity refreshes when status refreshes are triggered repeatedly.
- Adds a timeout to `rclone about` capacity probes so slow remotes do not hold the GUI refresh path indefinitely.
- Keeps mount status refresh immediate while allowing capacity information to update less aggressively.

## Verification

- `python3 -m py_compile $(find src build tests -name '*.py' -print) launcher.py`
- `PYTHONPATH=src python3 -m unittest discover -s tests -v`
