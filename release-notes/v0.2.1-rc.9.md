# SSH MountMate v0.2.1-rc.9

Pre-release build for capacity refresh performance.

## Fixes

- Uses the local mounted filesystem first for capacity display, matching Windows Explorer/macOS/Linux mountpoint capacity behavior when available.
- Avoids starting short-lived `rclone about` processes during normal mounted-drive capacity refreshes.
- Keeps remote capacity probes as a low-frequency fallback only when local mountpoint capacity cannot be read.
- Adds a lightweight 5-second local capacity refresh loop so usage bars can update without a full mount-status scan.

## Verification

- `python3 -m py_compile $(find src build tests -name '*.py' -print) launcher.py`
- `PYTHONPATH=src python3 -m unittest discover -s tests -v`
