# SSH MountMate v0.2.4-rc.1

Pre-release focused on high-DPI UI clarity and mount-card layout refinements.

## Changes

- Enable Windows process DPI awareness before creating the Tk window, reducing blurry system bitmap scaling on high-DPI displays.
- Apply a small Tk scaling increase to improve readability while preserving the existing layout balance.
- Raise card body/status text sizes slightly and improve muted-text contrast.
- Keep default Tk fonts at a clearer minimum size after language changes.
- Increase default/minimum window sizes for the main window, Settings, and Add config dialogs.
- Make toolbar/batch buttons, help icons, checkboxes, and browse buttons easier to read and click.
- Thicken the capacity usage bar.
- Further enlarge default dialog bounds and use slightly smaller bold action-button text for better balance on high-DPI displays.
- Increase help icon, checkbox, browse-button, and capacity-bar sizing so controls remain clear after packaging.
- Replace small native checkbox rendering with a larger custom checkbox and enlarge help icons/tooltips for a more balanced high-DPI UI.
- Scale default and minimum window sizes from Tk DPI scaling, while softening action-button and help-question mark weight.
- Refine help icon weight, action-button font weight, batch-import wording, and startup-mount wording.
- Restore the SSH config batch source label and shorten per-host batch import row summaries.
- Reduce default/minimum window widths, move capacity usage text below the bar, and show the capacity path inside a taller draggable capacity bar.
