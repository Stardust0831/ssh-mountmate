# SSH MountMate v0.2.3-rc.1

Pre-release focused on connection reliability reported in issue #5.

- Normalize automatically scanned host keys to the exact host/port marker rclone uses.
- Retry once without rclone `known_hosts_file` when a mount fails with `knownhosts: key mismatch`.
- Rebuild Windows ACLs for app-managed SSH config files to satisfy OpenSSH permission checks.
- Pass `--links` to `rclone mount` so remote symlinks do not abort the mount.
