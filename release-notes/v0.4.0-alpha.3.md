# SSH MountMate v0.4.0-alpha.3

This prerelease restores missing main-window operations and completes the settings usability audit
on top of `v0.4.0-alpha.2`.

## Main-window controls and logs

- Restores explicit **Mount all** and **Unmount all** buttons in the main window. The operations are
  disabled when no applicable connection is available.
- Adds **View log** to every connection card and a Mount logs section in Settings.
- Adds a read-only log window with refresh and copy-to-clipboard controls.
- Bounds the displayed log to its most recent 2 MiB and reports truncation, preventing an unusually
  large rclone log from making the GUI unresponsive.
- Handles connections whose log has not been created yet without presenting a file-read failure.

## Clear connection identity

- Mount, unmount, open, and remove status messages now use the user-visible connection name rather
  than the stable internal ID. A configuration renamed from `NAS` to `jzj` therefore reports `jzj`
  while retaining its compatible state and ID files.

## Settings controls

- Changes mount-at-login, automatic transfer popup, and automatic update checks from checkboxes to
  switch controls.
- Keeps cache mode and language as typed dropdown choices.
- Adds bilingual size and duration units with examples to cache limits, cache timings, write-back,
  and buffer-size inputs.
- Tests that file-manager integration settings appear only on supported desktop platforms.
- Does not change the settings schema; existing settings, state files, and custom rclone values
  remain compatible.

## Important prerelease limitations

- Packages are not yet production code-signed or notarized.
- macOS ARM64 retains the previously documented active-upload update timing risk.
- This release does not include an installer or the planned Experimental macOS NFS backend.
- The Rust rewrite PR remains Draft and is not merged by this release.

---

本预发行版在 `v0.4.0-alpha.2` 基础上恢复缺失的主窗口操作，并完成设置页可用性审查。

- 主窗口恢复“全部挂载”和“全部卸载”按钮。
- 每个连接卡片及设置页增加挂载日志入口；日志窗口支持刷新和复制到剪贴板。
- 日志最多显示最近 2 MiB，并提示内容已截断，避免超大日志拖慢界面。
- 挂载、卸载、打开和删除状态改为显示用户可见的配置名称，不再显示旧的内部 ID。
- 登录挂载、自动显示传输窗口和自动检查更新改为开关。
- 缓存大小、时长、回写延迟和缓冲区字段增加中英文单位及示例。
- 设置 schema、旧状态文件和自定义 rclone 参数保持兼容。
- 本版本不包含安装器或实验性 macOS NFS 后端，PR 仍保持 Draft。
