# SSH MountMate v0.2.1-rc.6

Pre-release build for Windows login mount startup.

## Fixes

- Replaced Windows `schtasks` login mount setup with a current-user `HKCU\Software\Microsoft\Windows\CurrentVersion\Run` startup entry.
- Windows startup now uses a single `--mount-startup-all` entry that mounts saved configs in batch after login.
- Avoids the `schtasks` "Access is denied" failure on systems where normal users cannot create scheduled tasks.

## Verification

- `python3 -m py_compile $(find src build tests -name '*.py' -print) launcher.py`
- `PYTHONPATH=src python3 -m unittest discover -s tests -v`
