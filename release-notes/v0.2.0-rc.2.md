# SSH MountMate v0.2.0-rc.2

This prerelease fixes the connection card action-button layout at narrower window widths.

## What's Changed

- Keep the four card action buttons pinned to the right side of each connection card.
- When the buttons switch from one row of four to a 2x2 layout, stop the unused third and fourth grid columns from reserving space.
- Give the connection text/capacity area more room in the narrow 2x2 button layout.

# SSH MountMate v0.2.0-rc.2 中文说明

这个预发布版修复较窄窗口下连接卡片操作按钮的布局问题。

## 主要更新

- 四个连接卡片操作按钮始终保持在卡片最右侧。
- 当按钮从一行四列切换为两行两列时，不再让未使用的第 3/4 列继续占位。
- 在较窄的两行两列按钮布局下，为连接信息和容量条留出更多空间。
