# SSH MountMate v0.2.2-rc.1

Pre-release build for smoother GUI refreshes.

## Fixes

- Reuses server card widgets during status and capacity refreshes instead of destroying and recreating every card.
- Updates card labels, buttons, colors, and capacity bars in place to reduce visible flicker.
- Reorders cards only when the saved config order changes.
- Forces the card canvas to settle after mouse-wheel scrolling to reduce residual drawing artifacts.

## Verification

- `python3 -m py_compile $(find src build tests -name '*.py' -print) launcher.py`
- `PYTHONPATH=src python3 -m unittest discover -s tests -v`
