# SSH MountMate v0.2.1-rc.10

Pre-release build for mount status handling.

## Fixes

- Removes the user-facing stale/expired mount state from the GUI.
- Treats a recorded running rclone PID or a ready mountpoint as mounted, avoiding a mismatch where the card looks stopped but mounting reports "already mounted".
- Allows Windows unmount to continue when the rclone command line cannot be read but the recorded PID is still running.

## Verification

- `python3 -m py_compile $(find src build tests -name '*.py' -print) launcher.py`
- `PYTHONPATH=src python3 -m unittest discover -s tests -v`
