# SSH MountMate v0.4.4-alpha.2

This prerelease makes batch connection organization match the intended workflow.

- In **Batch actions**, select connections and then add an existing tag from the picker or create
  a new tag in the adjacent field. When both are present, the newly entered tag is used.
- Batch login auto-mount is a single stateful checkbox. It stays clear for mixed selections and
  skips interactive SSH connections that require a live login session.
- **Manage tags** is now dedicated to global tag cleanup. Use the small x beside a tag to remove
  it from every connection.

---

本预发布版调整了批量连接整理交互，使其符合预期工作流。

- 在“批量操作”中，先选择连接，再从下拉列表选择已有标签，或在相邻输入框中新建标签后添加；若同时
  存在两者，以新输入的标签为准。
- 批量登录自启改为一个反映当前状态的复选框；混合选择不会误显示为已开启，要求实时登录会话的交互式
  SSH 连接会被跳过。
- “标签管理”现在只负责全局整理。点击标签旁的小 x，可从所有连接中移除该标签。
