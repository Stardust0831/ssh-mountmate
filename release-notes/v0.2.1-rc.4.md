# SSH MountMate v0.2.1-rc.4

Pre-release build for diagnosing Windows login mount setup failures.

## Fixes

- Captures `schtasks` stdout and stderr when Windows login mount task creation fails.
- Startup setup error dialogs now include the config display name, internal config id, command line, exit code, and Windows command output.

## Verification

- `python3 -m py_compile $(find src build tests -name '*.py' -print) launcher.py`
- `PYTHONPATH=src python3 -m unittest discover -s tests -v`
