# SSH MountMate v0.2.2

This release focuses on GUI refresh behavior and smoother mount card updates.

- Reuse mount card widgets during refreshes instead of rebuilding the full list.
- Avoid repeated no-op widget updates while refreshing mount status and capacity text.
- Keep card text widths stable between real layout changes to reduce horizontal jumps.
- Reuse capacity bar canvas items instead of clearing and redrawing the whole bar.
- Skip drawing capacity bars while their width is not ready yet.
- Remove forced synchronous canvas redraws during mouse-wheel scrolling.
