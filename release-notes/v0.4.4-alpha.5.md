# SSH MountMate v0.4.4-alpha.5

This prerelease corrects the search and sort row layout.

- The sort picker now fills its allocated responsive column instead of sizing only to its current
  label, removing the unexplained empty area on the far right.
- The saved-order label uses the same fill width while reordering, so entering and leaving reorder
  mode no longer shifts the row layout.

---

本预发布版修正了搜索与排序这一行的布局。

- 排序下拉框现在会填满分配给它的响应式区域，不再只按当前文字宽度显示，因此移除了最右侧意义不明
  的空白。
- 调整顺序模式下的“保存顺序”显示使用相同宽度，进入或退出排序模式时不会再造成这一行的布局跳动。
