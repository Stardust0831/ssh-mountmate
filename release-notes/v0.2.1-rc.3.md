# SSH MountMate v0.2.1-rc.3

Pre-release build for Windows mount status testing.

## Fixes

- Fixed Windows mount detection when SSH MountMate runs its bundled rclone from the managed app bin directory.
- Recognizes both normal `rclone.exe` processes and managed bundled `rclone-<hash>.exe` processes.
- Uses the same process detection for cleanup, so active managed rclone binaries are not mistaken for unused files.

## Verification

- `python3 -m py_compile $(find src build tests -name '*.py' -print) launcher.py`
- `PYTHONPATH=src python3 -m unittest discover -s tests -v`
