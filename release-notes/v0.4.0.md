# SSH MountMate v0.4.0

This is the first stable native Rust release of SSH MountMate for Windows, macOS, and Linux.

## Highlights

- Native x64 and ARM64 packages with the official rclone v1.74.4 and no Python runtime.
- Manual, SSH config, batch import, SAI, native SFTP, and external OpenSSH connection paths.
- Verified host-key pinning, safe process ownership, mount readiness, refresh, and unmount behavior.
- Full VFS write-back visibility with queued and active upload progress, transfer windows, and
  warnings before unsafe unmounts.
- Configurable simultaneous VFS uploads: 4 by default, presets for 8 and 12, and custom values from
  1 through 32.
- Tray or menu-bar controls, native notifications, taskbar or Dock progress, file-manager commands,
  login mounting, and batch mount or unmount actions.
- Verified in-app updates with SHA-256 validation, startup health confirmation, rollback, and
  preservation of active mounts and queued uploads while the GUI package is replaced.
- Migration of existing SSH MountMate connections, settings, rclone configuration, state, cache,
  and logs.

## Distribution notes

- The release contains six portable ZIP packages and `SHA256SUMS.txt`.
- Packages are not yet production code-signed or notarized. Windows and macOS may show their normal
  trust prompts for unsigned applications.
- This release does not include an installer, server changes, or the Experimental macOS NFS backend.

这是 SSH MountMate 首个稳定的原生 Rust 正式版，支持 Windows、macOS 和 Linux。

## 主要内容

- 提供 x64 和 ARM64 原生包，内置官方 rclone v1.74.4，不包含 Python 运行时。
- 保留手动配置、SSH config、批量导入、SAI、rclone 原生 SFTP 和外部 OpenSSH 连接方式。
- 验证主机指纹固定、进程所有权、挂载就绪、刷新和卸载行为。
- 显示 VFS 真实写回队列与上传进度，并在仍有上传时阻止误取消挂载。
- 可配置同时上传文件数：默认 4，可选 8、12，或自定义 1 至 32。
- 支持托盘或菜单栏、系统通知、任务栏或 Dock 进度、文件管理器命令、登录挂载和批量操作。
- 自更新会校验 SHA-256、执行启动健康确认和失败回滚；替换 GUI 时已有挂载和排队上传继续运行。
- 兼容迁移既有连接、设置、rclone 配置、挂载状态、缓存和日志。

## 发行说明

- 正式版包含六个平台 ZIP 和 `SHA256SUMS.txt`。
- 当前仍未进行生产代码签名或公证，Windows 和 macOS 可能显示未签名应用的系统提示。
- 本版本不包含安装器、服务器修改或实验性 macOS NFS 后端。
