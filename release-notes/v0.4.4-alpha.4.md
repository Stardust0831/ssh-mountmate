# SSH MountMate v0.4.4-alpha.4

This prerelease keeps all tag assignment and cleanup in one dedicated workflow.

- **Batch actions** now focuses on mounting, unmounting, per-connection login startup, and deleting
  connections. Tag assignment controls no longer appear there.
- **Manage tags** supports selecting one or multiple connections, including select-all, and then
  assigning either an existing tag or a newly entered tag.
- Per-connection tag removal and global tag deletion remain in tag management, so tag creation,
  assignment, and cleanup are handled in one place.

---

本预发布版将标签分配与整理集中到一个独立工作流中。

- “批量操作”现在专注于挂载、卸载、逐连接登录自启和删除连接，不再显示标签分配控件。
- “标签管理”支持选择单个或多个连接，也支持全选；选择后可以添加已有标签或新输入的标签。
- 从单个连接移除标签和全局删除标签仍保留在标签管理中，因此标签创建、分配和清理都在同一处完成。
