# SSH MountMate v0.2.1-rc.5

Pre-release build for managed bundled rclone dependency detection.

## Fixes

- Treats managed bundled rclone binaries such as `rclone-<hash>.exe` as valid rclone installs during dependency checks.
- Keeps dependency detection consistent with Windows mount status detection.

## Verification

- `python3 -m py_compile $(find src build tests -name '*.py' -print) launcher.py`
- `PYTHONPATH=src python3 -m unittest discover -s tests -v`
